 > Protocolo HTTP - HyperText Transfer Protocol - É UM PROTOCOLO DE COMUNICAÇÃO E TRANSFERÊNCIA DE DADOS PELA WEB
 Definido pelos RFCs 1945 e 2116
 Protocolo de comunicação - rege e estrutura as mensagens
 Browser - Implementa o cliente HTTP
 Servidor - Host objetos web
> O cliente envia mensagens do tipo Request HTTP 
> O servidor responde com mensagens do tipo Response HTTP
> Protocolo TCP - protocolo da camada de transporte, opera de duas maneiras:
conexão persistente: uma vez q estabeleço a conexão, ela será mantida, até que ocorra um pedido específico de finalização
conexão não persistente: a cada envio de mensagem, essa conexão é encerrada
Utiliza a arquitetura client-server - segmenta os papéis
Stateless - o servidor não armazena o estado do cliente, apenas responde com os dados que o cliente solicitou
